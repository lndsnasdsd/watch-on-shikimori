Политика конфиденциальности

Для расширения Watch on Shikimori
Последнее обновление: 15 апреля 2026 г.

Эта Политика конфиденциальности описывает, какие данные обрабатывает расширение Watch on Shikimori, как эти данные используются, кому они могут передаваться и какие варианты доступны пользователю.

1. Назначение расширения

Watch on Shikimori добавляет на поддерживаемые страницы Shikimori встроенный плеер для просмотра аниме, быстрые ссылки на связанные внешние ресурсы и дополнительные элементы интерфейса, связанные с просмотром эпизодов и навигацией по аниме.

2. Какие данные обрабатываются

Расширение может обрабатывать только те данные, которые необходимы для его основной функции:

  • URL и содержимое поддерживаемых страниц Shikimori, на которых работает расширение.
  • Технические данные о тайтле, эпизодах, переводах и связанных ресурсах.
  • Выбранный пользователем домен Shikimori и локальный кэш данных, сохраняемый в хранилище браузера.
  • Если пользователь вошел в аккаунт Shikimori в браузере, расширение может получать связанные с аккаунтом данные, необходимые для его функций, например идентификатор пользователя, списки аниме, статус просмотра и прогресс по эпизодам.

3. Какие данные расширение не собирает

Расширение не предназначено для сбора следующих категорий данных:

  • пароли, PIN-коды, секретные вопросы и иные учетные данные;
  • финансовая и платежная информация;
  • медицинская информация;
  • личные сообщения, электронная почта или чаты;
  • точные данные о местоположении.

4. Как используются данные

Данные используются исключительно для работы и улучшения основных функций расширения, включая:

  • определение, что открыта поддерживаемая страница Shikimori;
  • отображение встроенного плеера и связанных элементов интерфейса;
  • поиск информации об эпизодах, переводах и связанных материалах;
  • загрузку пользовательских списков и прогресса просмотра на Shikimori, если пользователь авторизован на сайте;
  • сохранение локальных настроек и кэша для более быстрой работы расширения.

5. Кому передаются данные

У расширения нет собственного сервера для сбора пользовательских данных. Разработчик не получает и не хранит копии пользовательских данных на собственных серверах.

Для выполнения своих функций расширение может отправлять запросы напрямую из браузера пользователя к сторонним сервисам, используемым самим расширением:

  • Shikimori — для получения данных о тайтлах, аккаунте, списках и прогрессе просмотра;
  • Kodik и связанным с ним доменам — для поиска и загрузки данных плеера и эпизодов;
  • AniLibria и связанным с ней API/ресурсам — для получения связанных данных и ссылок.

Такая передача происходит только в той мере, в какой это необходимо для работы основной функции расширения.

6. Хранение данных

Часть данных хранится локально в браузере пользователя через API хранения браузера, например:

  • выбранный домен Shikimori;
  • локальный кэш списков и метаданных;
  • служебные настройки расширения.

Эти данные остаются в браузере пользователя до их удаления самим пользователем, очистки данных расширения или переустановки расширения.

7. Продажа и передача данных

Разработчик не продает пользовательские данные и не передает их третьим лицам для рекламы, профилирования, оценки кредитоспособности или брокерской перепродажи данных.

Пользовательские данные не используются и не передаются в целях, не связанных с работой или улучшением основных функций расширения.

8. Безопасность

Расширение использует стандартные механизмы браузера и HTTPS-запросы к поддерживаемым сервисам там, где это применимо. Тем не менее ни один способ передачи или хранения данных не может гарантировать абсолютную безопасность.

9. Права пользователя и управление данными

Пользователь может:

  • удалить расширение в любое время;
  • очистить локальные данные расширения через интерфейс браузера или встроенные функции очистки кэша;
  • выйти из аккаунта Shikimori, если не хочет, чтобы расширение получало данные, связанные с авторизованным сеансом.

10. Изменения политики

Эта Политика конфиденциальности может обновляться. Актуальная версия должна публиковаться по тому же URL, который указан на странице расширения.

11. Контакты

Если у вас есть вопросы по этой Политике конфиденциальности, замените этот абзац на актуальные контактные данные, например адрес электронной почты разработчика или ссылку на страницу поддержки.

12. Заявление о соблюдении правил Chrome Web Store

Данные, обрабатываемые расширением, используются только для предоставления и улучшения его основной функции. Они не продаются и не передаются для показа персонализированной рекламы, перепродажи данных или целей кредитования.
